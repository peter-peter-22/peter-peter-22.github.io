Shop

Starting page

The first page the user ecounters introduces the shop, shows a few products and lists the top discounts.

Browsing

In the browsing menu the displayed products can be filtered by category, properties and discounts.

Authentication

The user can be authenticated with an email address or google auth. The session ends when the browser is closed unless the "remember me" option is checked. Sessions persist between server restarts.

Reviews

The users can rate a product and tell their opinions about it, or reply to a rating. The average rating is displayed at each product.

Checkout

The page uses stripe to handle the payments. The users are automatically registered to stripe and they can view their receipts in the user menu.

Editor UI

The categories, filters, products and discounts can be edited on the page without coding.

Flexible

The grid supports both portrait and landscape images.

Photo Gallery

Stylized

The page was designed to look nice. It also offers an adjustable preview for each category.

Editable

Images can be uploaded and the settings can be adjusted on the admin UI without coding.

This page was made with nodeJS, jquery with postgresql. In the administrator menu, the text fields on the main page, the images, the categories, the logo, the profile, the password and the display mode of the previews can be altered. The images are stored on the server as files and their url is saved into the database. The page is protected from SQL injection, only the password hash is stored and all admin tools require authentication.
